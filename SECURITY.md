# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project or on our websites, please report it responsibly.

**Email:** [info@regenstudio.world](mailto:info@regenstudio.world)

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact

## What to Expect

- We will acknowledge your report within 48 hours
- We will work with you to understand and resolve the issue
- We will not take legal action against good-faith security researchers

## Scope

- [regenstudio.world](https://www.regenstudio.world) and [demos.regenstudio.world](https://demos.regenstudio.world)
- Code in this repository

## Out of Scope

- Social engineering (phishing, pretexting)
- Denial of service attacks
- Issues in third-party dependencies (please report those upstream)
- Automated scanning without prior coordination
